AI GYM Coach project.
It is a python - telegram API based project with Groq API Integration.
UI available in two languages Ukrainian and English.
Accounts system available with registration process and editing details functions. 


SETTING UP:

after copiing project to your mashine use this steps:

1) git clone https://github.com/Alexukk/AiGymCoach.git
2) cd AiGymCoach
3) pip install requirements.txt
4) Create .env file in bot directory.
5) Add fields as shown below:
   GROQ_API_KEY=YOUR_API_GROQ
   BOT_TOKEN=YOUR_TELEGRAM_API

Get your Telegram API from BotFather in telegram
Get your Groq API from https://groq.com/
